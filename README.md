# Thiệp Hoa (Banner Designer)

Công cụ thiết kế và in băng rôn, thiệp chúc mừng hình Ovan cho lẵng hoa. 

## Tính năng nổi bật
- Web-based 100% Client-side.
- Tự động co giãn chữ vừa khít khung Ovan.
- Hỗ trợ in ấn chuyên nghiệp (A4 Landscape).

## Hướng dẫn cài đặt
1. Cài đặt các thư viện: `npm install`
2. Chạy môi trường Dev: `npm run dev`
3. Build bản Production: `npm run build`
